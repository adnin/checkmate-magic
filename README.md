**Checkmate Magic** is a top-down pixel ARPG where **every move is a spell**.  
You build your power using chess-inspired abilities and card-based upgrades, then fight through endless waves of enemies on an enchanted board.

---

## üß≠ Game Journey / Flow

### 1. Start of a Run

1. **Launch the game**  
   - You start at the **Main Menu**.
   - Options: `New Game`, `Continue` (if available), `Exit`.

2. **New Game ‚Üí Run Setup**  
   - Choose your **starting piece/build** (e.g. Knight-focused, Rook-focused, etc.).
   - Equip any unlocked **armor / passive items**.
   - Confirm to enter the arena.

---

### 2. In-Game Loop

A typical run goes through this loop:

1. **Spawn in the Arena**
   - You appear on a **tile-based battlefield** (chess-like grid).
   - Basic **movement** and **attack** are available immediately.

2. **Waves of Enemies**
   - Enemies spawn in **waves**.
   - Each wave gets progressively harder:
     - More enemies
     - New enemy types
     - Stronger stats / new attack patterns

3. **Combat & Abilities**
   - Move to dodge hits and reposition.
   - Use normal attacks + **special abilities** (inspired by chess pieces).
   - Element tiles (Fire / Ice / Lightning / Nature / etc.) can affect the battle:
     - Fire tiles: damage over time
     - Ice tiles: slow or freeze
     - Others: buffs / debuffs / control effects

4. **Rewards After Waves**
   - When you clear a wave, you get **rewards**, such as:
     - **Cards** (new abilities / modifiers)
     - **Weapons / Armor / Passives**
     - **Currency** for upgrades

5. **Card / Upgrade Phase**
   - Between waves, you can:
     - **Attach cards** to weapons or armor
     - Choose between **multiple upgrade options**
     - Plan synergies for your **next waves**

6. **Repeat the Loop**
   - `Wave ‚Üí Rewards ‚Üí Upgrade ‚Üí Next Wave`
   - Difficulty ramps up over time.

---

### 3. Boss & End of Run

1. **Boss Waves**
   - After certain waves, a **boss** appears.
   - Bosses have:
     - Higher HP
     - Unique attack patterns
     - Special mechanics (e.g. summons, arena hazards)

2. **If You Win**
   - You gain:
     - Bigger rewards
     - Unlocks (new cards, items, or cosmetics)
     - Progress towards long-term goals (meta progression)

3. **If You Die**
   - The run ends.
   - You return to the main menu / hub.
   - You may keep some **meta progression** (depending on your design):
     - Permanent stats
     - New cards added to the pool
     - New piece builds unlocked
